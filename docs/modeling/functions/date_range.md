Generates a range of dates between two dates.

### Syntax

 ```yaml
date_range(<range>)
```
    
| Parameter   | Type | Description |
| ----------- | ---- | ----------- |     
| range | Text | The range of values. |

### Return values
Array


## Usage

Generates a range of dates for the specified interval.


```yaml
date_range(MONTH)
```
