Evaluates an expression in a context modified by filters.
### Syntax

 ```yaml
keep_filters(<expr>)
```
    
| Parameter   | Type | Description |
| ----------- | ---- | ----------- |     
| expr | any | The expression to be evaluated. |

### Return values
Depends on input arguments.
