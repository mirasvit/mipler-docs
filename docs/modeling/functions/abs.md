Returns the absolute value of a number.

### Syntax

 ```yaml
abs(<expr>)
```
    
| Parameter   | Type | Description |
| ----------- | ---- | ----------- |     
| expr | Number | The number of which to return the absolute value. |

### Return values
Number


## Usage

Returns the absolute value of the discount amount.


```yaml
abs(orders.discount_amount)
```
